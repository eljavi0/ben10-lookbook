## Concepto de Diseño
El diseño sigue una estética minimalista oscura elegante, utilizando una grilla de 6 columnas para generar jerarquía visual y dinamismo.

## Paleta de Colores
Se utilizó verde como color acento representando el Omnitrix, sobre fondo oscuro para generar contraste.

## Jerarquía Visual
El título principal ocupa todo el ancho del grid, mientras que las secciones principales se distribuyen en 4 columnas y el aside en 2 columnas.

## Tecnologías usadas
- HTML5 Semántico
- CSS Grid
- Variables CSS
- Google Fonts
